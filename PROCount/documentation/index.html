Thank you for purchasing PROCount. In this documentation you will find details on how to customize the template for your needs. It's fairly simple actually, template contains Countdown Clock which requires jQuery. Also, you will find LESS file that I used to code this beautiful template, its simple and very easy to read.

Assets

Here are the description of the files and folders you find inside of an /assets directory;

/javascript/main.js - The main Javascript file that triggers Countdown and Subscription functions.

/stylesheet/style.less - This is a LESS formatted CSS StyleSheet, it is neat and easy to read. It's recommend you use LESS file and compress it into style.css. You can use free compressor tools like, SimpLESS.

Countdown Plugin

The countdown for demo sake comes with futureDate function which adds +60 days to current date, so you can remove the function.

To change the countdown clock open up a /assets/javascript/main.js file and go down to line 19:

date: futureDate()+' 12:00:00', // CHANGE THE DATE

Your date has to be in a format below:

MM/DD/YYYY HH:MM:SS

Here is an example:

date: '12/24/2012 12:00:00',

You also need to set your UTC offset. You can find your UTC offset here. Currently the offset is set to NYC -4 UTC. 

offset: -4 // CHANGE UTC OFFSET

Subscription Box

Subscription box is integrated with MailChimp. To get the form running, you just need your MailChimp API Key and List ID.

To get your API Key, click here.
To get your List ID, click here.

After you get your List ID and API Key, go to /assets/email/mailchimp.php and change the lines 2 & 3.

	$apiKey = '55977b64122d41058d1edaf8326d0ff9-us4'; // Input your Mailchimp API Key
	$listId = 'e83a30cc73'; // Input your List ID
